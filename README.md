# Simple-Calculator-Web-App
A responsive calculator web application built using HTML, CSS and JavaScript.
The application performs basic arithmetic operations and also allows users to switch between **Light Mode and Dark Mode**.

## Features
- Addition, Subtraction, Multiplication, Division
- Clear display functionality
- Light Mode and Dark Mode switching
- Interactive UI using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open *index.html* in a browser
